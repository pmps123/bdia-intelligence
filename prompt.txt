# Rencana Implementasi: Restrukturisasi Notion Clone Premium - BDIA Intelligence

Mengubah tata letak dan fungsionalitas UI aplikasi agar sepenuhnya menyerupai Notion Clone premium, menyatukan halaman Notes/Page, memfasilitasi visualisasi tabel ganda melalui Dropdown/Toggle, melakukan pemuatan data salesman secara dinamis dari file spreadsheet di root, menambahkan pratinjau (preview) Executive Report, dan menerapkan copywriting profesional dengan navigasi menu / (slash command).

## User Review Required

> [!IMPORTANT]
> - Penggabungan konsep **Page** dan **Notes** berarti halaman `/workspace` dan `/notes` akan disatukan. Kita akan mengarahkan semua ke satu halaman terpadu (misalnya `/workspace` atau `/notes`) yang memiliki daftar halaman bersarang di sidebar kiri.
> - Data Salesman akan dibaca langsung dari file Excel `Mapping Salesman Berjalan *.xlsx` yang ada di root saat inisialisasi aplikasi atau via tombol sinkronisasi sekali-klik.
